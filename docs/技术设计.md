# 技术设计

v1.0 以 v0.5 的规则引擎、预览引擎和 SQLite 数据层为基础，补充正式版发布能力和后续扩展所需的服务层。

```text
FileRenameAssistant
├─ Data
│  ├─ AppDatabase
│  ├─ RuleProfileRepository

├─ Models
│  ├─ FileItem
│  ├─ PreviewItem
│  ├─ RuleProfile

│  └─ ClassificationPlan
├─ Rules
├─ Services
│  ├─ PreviewEngine
│  ├─ ConflictDetector

```

## 环境检测

安装器不直接假设用户电脑已安装运行环境，而是执行随包脚本检测：

- `.NET 8 Desktop Runtime`
- `Windows App Runtime`

缺失时安装随包环境文件。

## 文件分类服务

v1.0 增加 `FileClassificationService`，用于后续 UI 调用。当前服务能根据扩展名把文件归类为：

- 图片
- 视频
- 文档
- 压缩包
- 音频
- 其它

该服务只生成分类计划，不直接移动文件。真实移动仍应走统一执行器和历史记录。

## 多字段排序模型

v1.0 增加 `MultiSortRule`，用于支持后续拖拽排序优先级。

当前 UI 仍保留单字段排序入口，但核心模型已准备好：

```text
排序字段 + 排序方向 + 优先级
```
