# 图标与按钮交互优化说明

本版本继续优化了图标、按钮和安装包视觉。

## 顶部标题

已移除左上角大标题下方的说明文字，避免遮挡右侧按钮。

## 保存规则按钮

已在规则名称输入框下方增加“保存规则”按钮，并绑定原有规则保存功能。

## 软件图标

新增统一软件图标：

```text
src\FileRenameAssistant\Assets\Icons\app.ico
src\FileRenameAssistant\Assets\Icons\app.png
```

并已接入：

- 编译后的软件 exe 图标
- Inno Setup 安装包图标
- 安装后的开始菜单/桌面图标

## 按钮图标

按钮已改成“图片 + 文字”组合，不再使用 emoji 字符。

图标统一放在：

```text
src\FileRenameAssistant\Assets\Icons
```

包括：

- 选择文件
- 选择文件夹
- 导出文件名
- 导入文件名
- 从列表移除
- 立即刷新
- 执行重命名
- 撤销
- 添加
- 删除
- 上移
- 下移
- 加载规则
- 保存规则

## 交互效果

按钮增加：

- 浅蓝圆角背景
- 阴影
- 鼠标悬浮背景变化
- 鼠标悬浮轻微上跳动画
- 按压反馈

预览列表保留渐入动画，执行重命名时保留动态进度条。
