# 本轮交互与安装包优化说明

## 文件与文件夹导入

顶部按钮现在按以下规则工作：

- `选择文件`：只把选中的文件追加到列表，不导入文件夹。
- `选择文件夹`：只把选中的文件夹本身追加到列表，不导入文件夹内文件。
- 已在列表中的路径不会重复加入。

文件夹也可以作为重命名对象执行改名。

## 手动修改新文件名

预览列表中的 `新文件名（双击可改）` 支持双击编辑。

执行重命名时：

- 如果选中了列表中的某几项，只重命名选中项。
- 如果没有选中任何项，默认重命名全部列表项。
- 重命名完成后，列表中的原文件名会刷新为新名称。

## 安装包

安装包继续使用 Inno Setup，不使用 MSIX。

安装包已改为：

- 中文安装包名称：`文件重命名助手_1.0.0_安装包.exe`
- 中文安装界面
- 中文开始菜单项
- 中文卸载项

卸载时会清理：

- 安装目录
- 本地数据目录：`%LOCALAPPDATA%\FileRenameAssistant`
- 安装脚本写入的注册表项：`HKCU\Software\文件重命名助手`

## 图标与动效

按钮已加入统一的图标文字风格，保留功能文字。

界面新增：

- 按钮圆角浅蓝背景
- 按钮悬停和按压反馈
- 预览列表行渐入动画
- 执行重命名时的动态进度条

图标风格参考了蓝色、浅色、圆角、文件处理类图标方向。
