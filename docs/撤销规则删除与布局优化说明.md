# 撤销、规则删除与布局优化说明

## 撤销修复

修复了单独重命名某个文件后点击撤销，列表仍然按修改后的文件名查找，导致显示：

```text
原文件不存在，可能已被移动或删除
```

现在撤销完成后会：

1. 从当前列表移除修改后的路径
2. 把恢复后的原路径重新加入列表
3. 刷新预览
4. 保留“撤销完成”的提示

## 删除规则

在“加载规则”按钮旁新增：

```text
删除规则
```

可删除当前选中的已保存规则。

## 布局调整

已按要求调整：

- 状态摘要移动到排序条件卡片下方
- 文件预览窗口占底部约三分之一高度
- 排序卡片的“添加条件、删除条件”改为并列
- 重命名卡片的“添加、删除”改为并列
- 规则名称和保存规则移动到“添加、删除”下方
- 排序条件每一行右侧集成上移/下移按钮
- 重命名规则每一行右侧集成上移/下移按钮
- 缩小按钮、图标和规则名称输入框，减少显示不全问题
- 提高按钮层级，减少悬浮动画被背景遮住的问题
