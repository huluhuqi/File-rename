# 规则链使用说明

规则链让你把多个简单规则按顺序串联，得到复杂效果。

## 典型场景

### 场景一：实验图片标准化

```
步骤 1：查找替换 IMG → 实验图
步骤 2：按修改日期排序
步骤 3：自动编号 _001
```

最终结果：

```
实验图_001.jpg
实验图_002.jpg
实验图_003.jpg
```

### 场景二：照片归档

```
步骤 1：命名模板 {mdate}_{name}{ext}
步骤 2：大小写转换 全小写
```

最终结果：

```
2025-06-01_dsc1001.jpg
2025-06-01_dsc1002.jpg
```

## 注意事项

- 规则按顺序执行，前一步的输出作为下一步的输入。
- 自动编号规则建议放在最后，避免被后续规则修改。
- 排序属于全局阶段，发生在规则链之前。
